# Conway's Game of Life

A simple <a href="https://codelabs.developers.google.com/your-first-webgpu-app" _target="blank">Google Codelab Project for learning WebGPU</a>.

## How to run

1. Download or clone the repo
2. Make sure you have <a href="https://nodejs.org/en/download" _target="blank">Node.js</a> installed.
3. Install dependencies using: `bash npm i`.
4. Run using `npm run dev`

### NOTE: Original Codelab uses pure HTML, JS, but this is done in React.js
